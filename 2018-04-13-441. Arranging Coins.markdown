思路2：每次循环总数n减去行数i+1，知道n小于零位置返回当前行数


python代码 (1108ms)

	class Solution(object):
    def arrangeCoins(self, n):
        """
        :type n: int
        :rtype: int
        """
        if n == 1:
            return 1
        if n == 0:
            return 0
        for i in range(n):
            n -= i+1
            if n < 0:
                return i
